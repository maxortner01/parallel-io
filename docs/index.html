Building

Currently, this software has three dependencies: Exodus II, Parallel NetCDF, and MPI. Each of these dependencies are expected to be easily found by the CMake using find_package.

This software was made to be easily used with spack. Simply use 

* spack env create pio
* spack env activate pio
* spack add exodusii parallel-netcdf mpi
* spack install

Then proceed to build in cmake as usual.

Information

Currently only ExodusII and NetCDF files are supported. Each file implementation uses template arguments to specify the access privileges which limits methods inside the implementation that can be used via SFINAE. Exodus file functionality is handled in an instance of pio::exodus::file whereas the NetCDF file and the parallel-io along with that is handled in pio::netcdf::file.

Warning
The exodus file implementation does not do any parallel IO currently. Becasue of this, the creation/manipulation of an instance of pio::exodus::file should be done in one process (like the root process for example). NetCDF files do not have this restriction.
